# ============================================
# RESUMO - AULA 1: LÓGICA DE PROGRAMAÇÃO E ALGORITMOS
# ============================================

# 🔹 CONCEITOS INICIAIS
# Lógica: forma racional de pensar e organizar ideias.
# Algoritmo: sequência de passos para resolver um problema ou executar uma tarefa.
# Exemplo: receita de bolo ou rotina diária.

# 🔹 SISTEMAS DE COMPUTAÇÃO
# Hardware: parte física que executa programas.
# Software: conjunto de instruções (algoritmos) que o hardware interpreta.
# Ambos formam o sistema computacional.

# 🔹 MÁQUINA DE VON NEUMANN
# Estrutura básica de qualquer computador moderno:
#   1. Memória – armazena dados e instruções.
#   2. ULA – realiza cálculos e operações lógicas.
#   3. UC – controla o fluxo de execução.
#   4. Entrada – recebe dados (ex: teclado).
#   5. Saída – mostra resultados (ex: monitor).
# ULA + UC = CPU (processador).

# 🔹 BIT, BYTE E PALAVRA
# Bit: menor unidade (0 ou 1).
# 8 bits = 1 byte.
# 1.024 bytes = 1 KB → 1 MB → 1 GB → 1 TB → 1 PB.
# CPU trabalha com blocos de bits (ex: 64 bits = 8 bytes = 1 palavra).

# 🔹 SISTEMA OPERACIONAL
# Software que gerencia hardware e outros programas.
# Exemplos:
#   - Computadores: Windows, Linux, macOS.
#   - Celulares: Android, iOS.
#   - Outros: FreeBSD, FreeRTOS (embarcados).
# Função: abstrair o hardware e facilitar o desenvolvimento de software.

# 🔹 REPRESENTAÇÕES DE ALGORITMOS
# 1. Descrição narrativa – linguagem natural, informal.
# 2. Pseudocódigo – estrutura lógica próxima da programação.
# 3. Fluxograma – representação gráfica do fluxo de execução.

# ============================================
# CONCLUSÃO
# A lógica é a base da programação.
# Algoritmos são instruções ordenadas para resolver problemas.
# Computadores executam algoritmos por meio da arquitetura de von Neumann.
