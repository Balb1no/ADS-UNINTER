# Resumo CSS - Display, Position e Z-index

1. Display
   - Define como o elemento é exibido na página.
   - Valores principais:
     - block → ocupa toda a linha (ex: <div>, <p>).
     - inline → ocupa só o espaço do conteúdo (ex: <span>, <a>).
     - inline-block → mistura os dois (fica na linha, mas aceita width/height).
     - none → oculta o elemento.
     - flex → ativa o modelo flexbox (layout flexível).
     - grid → ativa o modelo grid (layout em grade).

2. Position
   - Controla o posicionamento do elemento na página.
   - Valores principais:
     - static → padrão, segue o fluxo normal.
     - relative → posição relativa ao local original.
     - absolute → posição fixa dentro do contêiner pai (que tenha position diferente de static).
     - fixed → posição fixa em relação à janela (mesmo rolando a página).
     - sticky → mistura relative e fixed (fica preso ao rolar até certo ponto).
   - Propriedades usadas junto: top, right, bottom, left.

3. Z-index
   - Controla a ordem de sobreposição dos elementos.
   - Quanto maior o valor, mais “por cima” o elemento aparece.
   - Exemplo:
     .caixa1 { z-index: 1; }
     .caixa2 { z-index: 10; }
   ➡️ A caixa2 aparece sobre a caixa1.
