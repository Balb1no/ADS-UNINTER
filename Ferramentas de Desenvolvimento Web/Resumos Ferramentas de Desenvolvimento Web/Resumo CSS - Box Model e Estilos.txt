# Resumo CSS - Box Model e Estilos

1. Cores (color)
   - Define a cor do texto.
   - Exemplo: p { color: red; }

2. Background
   - Define o fundo do elemento.
   - Exemplo: div { background-color: lightblue; }

3. Bordas (border)
   - Define a linha ao redor do elemento.
   - Sintaxe: border: largura estilo cor;
   - Exemplo: div { border: 2px solid black; }

4. Margin
   - Espaço externo entre o elemento e outros.
   - Exemplo: div { margin: 20px; }

5. Padding
   - Espaço interno entre o conteúdo e a borda.
   - Exemplo: div { padding: 15px; }

6. Box Model
   - Todo elemento é uma “caixa” formada por:
     [margin] → [border] → [padding] → [conteúdo]
   - Controla como o espaço é calculado na página.

7. Width e Height
   - Definem largura e altura do conteúdo da caixa.
   - Exemplo: div { width: 200px; height: 100px; }
