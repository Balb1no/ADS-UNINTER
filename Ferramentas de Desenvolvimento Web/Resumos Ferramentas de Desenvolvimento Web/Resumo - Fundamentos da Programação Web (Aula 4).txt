# Resumo - Fundamentos da Programação Web (Aula 4)

## HTML5 - Elementos Estruturais Semânticos
- <header>: cabeçalho introdutório ou de navegação.
- <section>: seção temática de conteúdo.
- <article>: conteúdo independente (ex.: artigo, post).
- <nav>: links de navegação principais.
- <aside>: conteúdo lateral/tangencial (ex.: barra lateral).
- <footer>: rodapé com informações adicionais.
- <hgroup>: agrupamento de títulos/subtítulos.
- <div>: container genérico, sem valor semântico (último recurso).

## CSS Grid Layout
- Sistema bidimensional (linhas e colunas).
- Propriedades principais:
  - `display: grid;` → define o container.
  - `grid-template-columns` / `grid-template-rows` → define colunas e linhas.
  - `gap` → espaçamento entre linhas/colunas.
  - `grid-column` / `grid-row` → posicionamento dos itens.
  - `grid-area` → posicionamento por áreas.
- Funções úteis: `repeat()`, `calc()`, `span`, `minmax()`.
- Unidade `fr`: fração do espaço disponível.
- `grid-template-areas`: nomear regiões do layout.

## Flexbox
- Ideal para layouts unidimensionais (linha ou coluna).
- Propriedades principais:
  - `display: flex;` → define o container.
  - `flex-direction`: direção dos itens (row, column, reverse).
  - `justify-content`: alinhamento no eixo principal.
  - `flex-wrap`: quebra de linha dos itens.
  - `align-items`: alinhamento no eixo cruzado.
  - `align-content`: distribuição de múltiplas linhas.
