# Resumo CSS - Inline, Interno e Externo

1. CSS Inline
   - Estilo aplicado direto na tag HTML.
   - Usa o atributo style.
   - Exemplo:
     <p style="color: red; text-align: center;">Texto em vermelho</p>
   - Vantagem: rápido para testes.
   - Desvantagem: difícil de manter em projetos grandes.

2. CSS Interno
   - Estilo escrito dentro do próprio arquivo HTML.
   - Fica dentro da tag <style> no <head>.
   - Exemplo:
     <head>
       <style>
         p { color: blue; }
       </style>
     </head>
   - Vantagem: organiza melhor que inline.
   - Desvantagem: só vale para aquela página.

3. CSS Externo
   - Estilo separado em um arquivo .css.
   - Conectado ao HTML com <link>.
   - Exemplo:
     <head>
       <link rel="stylesheet" href="style.css">
     </head>
   - Vantagem: reaproveita estilos em várias páginas.
   - Desvantagem: precisa carregar o arquivo externo.
