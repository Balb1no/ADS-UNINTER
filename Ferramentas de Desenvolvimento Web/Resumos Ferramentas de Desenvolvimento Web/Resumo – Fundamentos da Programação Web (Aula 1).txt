Resumo – Fundamentos da Programação Web (Aula 1)

Introdução
- HTML é essencial para o desenvolvimento web.
- Objetivo: aprender tags básicas para criar e modificar páginas.
- Estrutura do curso:
  1. História do HTML e arquitetura cliente/servidor
  2. Ambiente de desenvolvimento (editores)
  3. Estrutura HTML, títulos, parágrafos, listas
  4. Links, imagens e tabelas
  5. Formulários

História do HTML
- HTML = HyperText Markup Language.
- Conceito de hipertexto: textos, imagens, sons, vídeos e links.
- Vannevar Bush (Memex, 1945) → links entre documentos.
- Ted Nelson (1960) → termo hipertexto/hipermídia, projeto Xanadu.
- Tim Berners-Lee (1990) → criação do HTML para troca de informações.
- Baseado no SGML. Evoluiu até o HTML5.

Termos atuais
- Front-end: interface gráfica (HTML, CSS, JS).
- Back-end: lógica, banco de dados, servidores (Java, PHP, .NET etc.).
- Full-stack: combinação de front-end e back-end.

Arquitetura Cliente/Servidor
- Cliente: dispositivo que acessa via navegador.
- Servidor: armazena sites e aplicativos.
- Comunicação via protocolo HTTP (requisições e respostas).

Ambiente de Desenvolvimento
- Bloco de Notas: básico, salva como .html.
- Notepad++: gratuito, mais recursos.
- Sublime Text: minimalista, rápido.
- Visual Studio Code: completo, multiplataforma, gratuito, com extensões e integração Git.
- Extensões úteis: Atom One Dark Theme, Winter is Coming Theme, Material Icon Theme, vscode-icons, Open in Browser, Live Server.

HTML – Estrutura e Tags
- HTML é linguagem de marcação interpretada pelo navegador.
- Arquivo salvo com extensão .html.
- Estrutura mínima:
  <!DOCTYPE html>
  <html lang="pt-br">
  <head>
    <meta charset="UTF-8">
    <title>Document</title>
  </head>
  <body>
  </body>
  </html>

Tags principais
- <h1> a <h6> → títulos (níveis de importância).
- <p> → parágrafo.
- <br> → quebra de linha.
- <hr> → linha horizontal.
- <ul> e <ol> → listas não ordenadas e ordenadas.
- <li> → itens da lista.
